info Account Roles Explained
Admin
  • Full access to all features
  • Add / edit / delete students
  • Manage servants
  • Import & export Excel data
  • Send announcements
  • Manage users and system/security settings
Sub-admin
  • See all classes and reports
  • Manage attendance, points, birthdays, announcements, and student data
  • Cannot manage accounts or system/security settings
User
  • Take attendance
  • View student profiles
  • View absence alerts
  • Submit Eftikad / Visitation reports
  • View birthdays and announcements
  • Cannot manage users or all classes
Rewards
  • Points page access only
  • Can add or deduct rewards points
  • Can be limited to selected grades and sections
  • Cannot take attendance, manage students, or use admin tools
